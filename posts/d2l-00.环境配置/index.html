前言

最近寒假在家想开始学深度学习，准备以李沐大神的这本书《动手学深度学习》开始。这学期对深度学习开始有了一点了解，觉得确实很有用，无论是思维上还是实际应用方面。

但是难在了第一步：配置环境，差点被劝退。其实官方给的配置很详细了

安装 — 动手学深度学习 2.0.0-beta0 documentation (d2l.ai)

安装 Miniconda

官网下载 64 位的 miniconda 即可，Miniconda — Conda documentation

安装过程一路默认即可，安装完打开 Anaconda Powershell Prompt (miniconda) ，将 conda 和 pip 的软件源改成清华的源，下载速度就会快很多

conda  config  --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda  config  --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda  config  --set show_channel_urls yes

pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

输入命令创建新的环境:

conda create --name d2l python=3.8 -y

注意，这里要先关闭 VPN 软件，不然会报错

conda activate d2l

安装深度学习框架和 d2l 软件包

我选择的是 PyTorch ，而且没有 GPU，就先学着吧，入门阶段就不考虑配置了。

pip install torch==1.8.1
pip install torchvision==0.9.1
pip install d2l==0.17.1
