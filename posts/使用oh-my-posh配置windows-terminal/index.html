使用Oh My Posh配置Windows Terminal

2021-06-29
1分钟阅读时长
过程记录

越看越觉得 Windows Terminal 的配置难看，就想着换个主题配色什么的，直接去官网看文档，官方推荐使用 Powerline，就按照文档一步步来，最终效果也还不错，权当记录一下配置过程。

安装 Powerline 字体

Powerline 使用字形来设置提示符样式。 如果你的字体不包含 Powerline 字形，则在整个提示符中，你可能会看到若干 Unicode 替换字符“&#x25AF”。 尽管 Cascadia Mono 不包括 Powerline 字形，但你可以安装 Cascadia Code PL 或 Cascadia Mono PL，这两者包含 Powerline 字形。 可以从 Cascadia Code GitHub 发布页安装这些字体。

翻译一下就是，上面的一些字符需要这个字体，推荐安装 Cascadia Mono PL 这个字体。

安装 Oh My Posh

打开 Power shell，依次输入命令安装 Posh-Git 和 Oh-My-Posh：

Install-Module posh-git -Scope CurrentUser
Install-Module oh-my-posh -Scope CurrentUser

自定义 Powershell 提示符

使用 notepad $PROFILE 或所选的文本编辑器打开 PowerShell 配置文件。 这不是你的 Windows 终端配置文件。 你的 PowerShell 配置文件是一个脚本，该脚本在每次启动 PowerShell 时运行。 详细了解 PowerShell 配置文件。

使用 vscode 打开脚本，脚本默认位置位于 ..\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1

code $PROFILE

输入以下内容：

Import-Module posh-git
Import-Module oh-my-posh
Set-PoshPrompt -Theme robbyrussel

每次打开 Windows Terminal 时运行这个脚本

也可以换 Oh My Posh 自带的其他主题：

Get-PoshThemes

想定制主题可以在 ..\Documents\WindowsPowerShell\Modules\oh-my-posh\3.166.3\themes 找到主题的配置文件，进行深度定制。

更换终端字体

打开 Windows Terminal 配置

选择安装的 Cascadia Code 字体，这个字体有点粗，字体粗细改为细体。

重新打开终端，就可以看到配置后的效果
