PicGo+Gitee图床搭建

2021-02-25
1分钟阅读时长
过程记录

本文参考 PICGO+Gitee搭建图床 ，无意间发现宝藏，于是实操，解决图片痛点，万分感谢。

前提

  • Gitee
  • PicGo
  • Node.js

创建 Gitee 仓库

创建公开仓库，选择单分支模型

生成私人令牌

配置 PicGo

安装 github-plus 插件，预先安装好 node.js

设置图床参数

  • repo 为创建仓库地址， 用户名+仓库名
  • token 为生成的私人令牌
  • origin 设为 gitee
  • 其他设为默认即可

通过 Typora 编辑 markdown 文件

打开偏好设置

设置插入图片时自动上传

将上传服务设定为 PicGo(app) ，选择安装路径，即可验证，若出现上传成功弹窗，即上传成功，在 gitee 仓库即可看到上传图片
