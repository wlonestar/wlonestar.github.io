Java的静态类、块、方法和变量

2022-01-04
2分钟阅读时长
学习笔记

明天早上考 Java，今天才开始复习（哈哈）。同学提了个关于 Java 里面 static 的问题，发现自己没怎么真正弄明白，遂求助谷歌，发现一篇文章，讲得很清楚，于是尝试翻译成中文。

Java - Static Class, Block, Methods and Variables (beginnersbook.com)

static 关键字可以用来修饰类、块、方法和变量。静态成员属于这个类而不是一个特定的实例对象，这意味着如果你用 static 修饰了一个成员，你可以在没有对象的情况下访问它。让我们通过一个例子来理解这一点。

这里我们有一个静态方法 myMethod() ，我们可以不通过对象直接调用这个方法，因为我们通过 static 关键字让这个成员提升到了类级别。如果我们移除 static 关键字让它成为非静态的，我们需要创建一个类的对象实例才能使用。

class SimpleStaticExample
{
    // This is a static method
    static void myMethod()
    {
        System.out.println("myMethod");
    }

    public static void main(String[] args)
    {
          /* You can see that we are calling this
           * method without creating any object. 
           */
           myMethod();
    }
}

输出：

myMethod

静态块

静态块用来初始化静态变量。当这个类被加载到内存中时这个块就被执行。一个类中可以有多个静态块，它们会按照在程序中的顺序执行。

例1：一个静态块

你可以看到这两个静态变量在被 main 方法调用之前就被初始化了。

class JavaExample{
   static int num;
   static String mystr;
   static{
      num = 97;
      mystr = "Static keyword in Java";
   }
   public static void main(String args[])
   {
      System.out.println("Value of num: "+num);
      System.out.println("Value of mystr: "+mystr);
   }
}

输出：

Value of num: 97
Value of mystr: Static keyword in Java

例2：多个静态块

多个静态块会按照它们在程序里的顺序执行。这就是为什么第一个代码块初始化的值会被第二个代码块覆盖。

class JavaExample2{
   static int num;
   static String mystr;
   //First Static block
   static{
      System.out.println("Static Block 1");
      num = 68;
      mystr = "Block1";
  } 
  //Second static block
  static{
      System.out.println("Static Block 2");
      num = 98;
      mystr = "Block2";
  }
  public static void main(String args[])
  {
      System.out.println("Value of num: "+num);
      System.out.println("Value of mystr: "+mystr);
   }
}

输出：

Static Block 1
Static Block 2
Value of num: 98
Value of mystr: Block2

静态变量

静态变量对于类的所有对象实例都是通用的，因为它是类级别的变量。换句话说静态变量只会被创建一次然后和所有类的对象实例共享。当类被加载到内存中时，对于静态变量，内存分配只发生一次。

两个重要的点：

  • 静态变量也被称为类变量

  • 不像非静态变量，静态方法和非静态方法可以直接访问静态变量

静态方法

静态方法可以在不使用类的对象实例的情况下访问静态变量，但是只能使用对象访问非静态方法和非静态变量。

静态方法可以在静态方法和非静态方法中直接访问。

静态类

一个静态类必须是一个嵌套类

  • 嵌套类不需要外部类的引用

  • 一个静态类不能获取外部类的非静态成员

class JavaExample {
   private static String str = "BeginnersBook";

   //Static class
   static class MyNestedClass{
    //non-static method
    public void disp() {

       /* If you make the str variable of outer class
        * non-static then you will get compilation error
        * because: a nested static class cannot access non-
        * static members of the outer class.
        */
       System.out.println(str); 
    }

   }
   public static void main(String args[])
   {
       /* To create instance of nested class we didn't need the outer
    * class instance but for a regular nested class you would need 
    * to create an instance of outer class first
        */
    JavaExample.MyNestedClass obj = new JavaExample.MyNestedClass();
    obj.disp();
   }
}

输出：

BeginnersBook
