使用 Git 创建版本库

初始化

输入命令：

mkdir <项目名称>
cd <项目名称>   #进入目录
git init       #初始仓库，新建一个 .git 目录
ls -la         #查看所有文件
touch <file1>  #创建一个文件
mkdir <dir1>   #新建一个目录

现在可以修改文件内容

git status -sb      #文件前面有 ?? 号
git add .           #一次性将文件添加到暂存区
git status -sb      #现在文件前面 ?? 变成了 A
git commit -m "xx"  #将你 add 过的内容正式提交到本地仓库，并添加注释
git log             #查看历史变动

文件变动

git status -sb
git add .
git commit -m "xx"

将本地仓库上传到 GitHub

新建一个空仓库，名称与本地目录名一致，复制ssh

不需要勾选任何东西

git remote add origin git@github.com:xxxx/<仓库名>
git push -u origin master

更改文件内容

git add .
git commit -m "xx"
git push
