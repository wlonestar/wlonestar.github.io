搭建hexo博客部署到Github

2021-02-23
2分钟阅读时长
过程记录

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装前提

  • git

  • node.js

git

  1. 下载安装：点击链接下载 git-scm.com

  2. 验证安装：win + R 打开运行，输入 cmd ，打开命令提示符

git --version

  1. 配置 ssh ：在桌面右键打开 git bash

ssh-keygen -t rsa -b 4096 -C "你的邮箱"

按回车三次

cat ~/.ssh/id_rsa.pub

得到一串东西，完整地复制这一串东西，打开 GitHub

点击右上角 头像 - settings - SSH and GPG keys，点击 New SSH key

随便起个名字，把之前复制地一串东西粘贴进去

回到 git bash

ssh -T git@github.com

....
Are you sure you want to continue connecting(yes/no)?

输入 yes ，回车

Hi xxx! You've successfully authenticated, but GitHub does not provide access.

就说明 ssh 配置好了

  1. 配置 git：打开 git bash ，依次输入
git config --global user.name 你的英文名
git config --global user.email 你的邮箱

将 ”你的英文名“ 和 ”你的邮箱“ 换成注册 github 所用的英文名和邮箱

这样 git 就配置好了

node.js

  1. 下载安装：点击链接下载 nodejs.org

  2. 验证安装：win + R 打开运行，输入 cmd ，打开命令提示符

node --version
npm --version

  1. 更换镜像源：国内镜像源速度较快

    打开 Powershell（管理员）

npm install -g cnpm --registry=https://registry.npm.taobao.org

等待安装完成

cnpm.cmd --version

利用 hexo 搭建博客

  1. 安装 hexo 博客框架：在powershell（管理员）输入
cnpm.cmd install -g hexo-cli

等待安装完成，输入

hexo.cmd --version

若出现依赖信息则安装完成

  1. 初始化博客：新建一个文件夹 blog，打开 cmd
cd D:\blog

初始化博客

hexo.cmd init
  1. 启动博客：
hexo.cmd s

表示博客在4000端口运行，打开浏览器，网址栏输入 loaclhost:4000，即可打开博客，按 Ctrl + C 终止博客

  1. 新建博客：

    在文件资源管理器打开 blog 文件夹下的 source 目录，再进入 _posts 目录，可以看到有 hexo 自动生成的 hello-world.md 文件，输入命令切换到当前目录，新建博客文章

cd ./source/_postshexo.cmd n "my first blog"

可以看到在_posts目录下出现了新建的 my-first-blog.md 文件

打开markdown文件，编辑保存，依次输入命令

hexo.cmd clean
hexo.cmd g
hexo.cmd s

即可在4000端口看到更改

  1. 将本地博客部署到 github 上：在 github 新建一个仓库，名称必须为 ”username“.github.io

    例如：wjl-lab.github.io

    回到 blog 目录下

cnpm.cmd install --save hexo-deployer-git

安装 git 部署的插件，等待安装完成

打开 blog 目录下的 _config.yml 文件，在文件最后更改 deploy

其中的 repo 地址为 GitHub 上创建的仓库地址

hexo.cmd d

会自动生成很多东西，提醒输入username 和 password 时，输入GitHub登录的账号密码即可

刷新GitHub仓库页面即可看到刚刚提交了commit，直接搜索 wjl-lab.github.io 即可访问博客

稍微配置一下即可得到不错的效果.
