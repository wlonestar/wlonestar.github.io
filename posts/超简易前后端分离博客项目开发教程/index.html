简易前后端分离博客项目开发教程

2021-08-22
1分钟阅读时长
开源项目

项目已开源，地址： wjl-lab/miniblog

miniblog

miniblog 是 tinyblog 的升级版，支持 markdown 语法

本项目是基于Spring Boot 和 Vue3 开发的一个前后端分离的简易博客项目

实现了创建、修改、删除文章操作

文章支持 markdown 语法

技术栈：

后端：

  • Spring Boot 2.x

  • Spring JPA

  • smart doc 生成 API 文档，文档的 HTML 版本在 doc 目录下

前端：

  • Vue 3.x
  • md-editor-v3 markdown 编辑器
  • vue-simple-markdown markdown 转 html 预览
  • bootstrap

构建运行

数据库

修改 application.xml 里的数据库配置，启动项目后表会自动生成

后端

项目打包 mvn clean package

在 target 包下运行 java -jar miniblog-0.1.0.jar

即在配置文件指定的 8081 端口启动了项目

前端

运行 npm run build

项目打包在 dist文件夹

安装简易HTTP 服务器 serve npm install serve -g

启动前端服务器 serve -s dist

即在默认的 5000 端口启动了项目

用浏览器打开，即可使用
