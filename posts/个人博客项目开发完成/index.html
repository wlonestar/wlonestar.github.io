Install

git clone https://github.com/wjl-lab/lblog

Usage

  • 数据库

    创建 lblog 数据库，修改 yml 配置文件，表会在项目启动时自动生成

    注意修改表 article 的 content 字段为 longtext 类型

  • 后端

$ cd blog-server
$ mvn clean package
$ cd target
$ java -jar lblog-0.0.1.jar
  • 博客页面
$ cd blog-app
$ npm install
$ npm run serve
  • 博客后台管理
$ cd blog-admin
$ npm install
$ npm run serve
