我就按照我目前学习到的内容写一篇教程，关于创建一个 spring boot web 项目的全过程。

新建数据库

打开MySQL命令行，输入命令新建数据库

create database demo00;

我这里就用了和项目相同的数据库名。

创建 springboot 项目

导入依赖

打开 IDEA，选择 Spring Initializr ，选择工程路径、名称等等。

这里先只勾选 Spring Web 依赖，其他依赖可以通过 Maven 导入。

就可以进入到项目中，修改Maven路径，改为之前配置的配置文件和本地仓库位置。

修改 pom.xml 文件导入相应依赖。

等待依赖加载完毕。

编写配置文件

把 src-main-resources 下的 application.properity 改成 application.yml

写入配置信息。

#服务器端口
server:
  port: 8080
#数据库配置
spring:
  datasource:
    type: com.alibaba.druid.pool.DruidDataSource
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/demo00?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=GMT&allowPublicKeyRetrieval=true
    username: root
    password: 123456
  thymeleaf:
    cache: false
#mybatis配置文件位置
mybatis:
  type-aliases-package: com.wjl.model
  mapper-locations: classpath:/mybatis/*.xml
#打印日志
logging:
  file:
    name: log/log.log
  level:
    root: info
    wjl: debug

在项目目录下新建 log 文件夹，File-Project Structure-Modules

连接数据库

打开右边的 Database

选择MySQL数据库

输入用户名密码和数据库名称，点击 Test Connection 会报错，那是因为新版本的MySQL必须要设置一下时区。

把时区设为 UTC ，再测试连接，即可成功。

我喜欢在 IDEA 里用 sql 脚本的方式创建数据表。

这样我们的数据库就创建完成了。

实现 MVC 三层模型

建包

右键 com.wjl ，选择 New-Package

新建 model 层、 mapper层、controller层

创建实体类

根据数据库的两个表创建对应的实体类，使用 lombok 注解自动实现 Construction、getter() and setter() 方法等。

// User.java
@Data
@NoArgsConstructor
@AllArgsConstructor
public class User {
    private Integer id;
    private String username;
    private String password;
    private String token;
}

// Comment.java
import java.util.Date;
@Data
@NoArgsConstructor
@AllArgsConstructor
public class Comment {
    private Integer id;
    private String username;
    private String comment;
    private Date time;
}

注意这里导入的包是 java.util.Date ，不是数据库的那个包。

实现 mapper 层

因为这个项目很简单，这里就直接通过注解操作数据库。

创建 UserMapper 、 CommentMapper 接口类。

// UserMapper.java
@Mapper
public interface UserMapper {
    // 注册成功，插入一条数据
    @Insert("insert into user(username,password,token) values (#{username},#{password},#{token})")
    void add(User user);
    // 检验注册的username是否已经存在数据库中，传进username进行与数据库比较
    @Select("select * from user where username = #{username}")
    User findByUsername(@Param("username") String username);
    //登录用：检验账号密码是否正确
    @Select("select * from user where username = #{username} and password = #{password}")
    User findByUsername_login(@Param("username") String username,
                              @Param("password") String password);
    @Select("select * from user where token = #{token}")
    User findByToken(@Param("token") String token);
    @Select("select username from user where username=#{username}")
    List<String> exitUsername();
    @Select("select * from user where username = #{username}")
    User username_to_setToken(@Param("username") String username);
} 

// CommentMapper.java
@Mapper
@Repository
public interface CommentMapper {
    // 发布留言，插入数据库
    @Insert("insert into comment (username, comment, time) values (#{username}, #{comment}, #{time})")
    void addComment(Comment comment);
    // 寻找username,为“发布留言”功能提供 username
    @Select("select * from user where username = #{username}")
    User findByUsername(@Param("username") String username);
    // 展示所有留言
    @Select("select * from comment")
    List<Comment> showAllComment();
    @Select("select comment from comment")
    List<String> listAllComment();
}

实现控制层

因为我们有三个不同页面，就对应创建三个控制器 LoginController 、 RegisterController 、 IndexController

// LoginController.java
@Controller
public class IndexController {
    @Autowired
    private UserMapper userMapper;
    @GetMapping("/")
    public String index() {
        return "/login";
    }
    @GetMapping("/toLogin")
    public String toLogin() {
        return "/login";
    }
    @PostMapping("/loginUser")
    public String login(
            @Param("username") String username,
            @Param("password") String password,
            HttpServletResponse response,
            Map<String, Object> map
    ) {
        User user = new User();
        user.setUsername(username);
        user.setPassword(password);
        User check_login = userMapper.findByUsername_login(username, password);
        if (check_login != null) {
            User user_token = userMapper.username_to_setToken(username);
            String token = user_token.getToken();
            response.addCookie(new Cookie("token", token));
            return "redirect:/homePage";
        } else {
            map.put("msg", "用户名或密码错误，请重新登录");
            return "/login";
        }
    }
}

// RegisterController.java
@Controllerpublic 
class RegisterController {    
    @Autowired    
    private UserMapper userMapper;
    
    @GetMapping("/toRegister")    
    public String toRegister() {        
        return "register";    
    }    
    
    @RequestMapping("/registerUser")    
    public String register(User user, Map<String, Object> map) {        
        user.setToken(UUID.randomUUID().toString());        
        List<String> userList = userMapper.exitUsername();        
        if (userList.contains(user.getUsername())) {            
            map.put("msg", "用户名[ "+user.getUsername()+" ]"+"已存在, 请重新注册");
            return "/register"; 
        } else {            
            User userCheck = userMapper.findByUsername(user.getUsername());
            if (userCheck == null) {                
                userMapper.add(user);                
                return "/login";            
            } else {                
                return "redirect:/homePage";            
            }        
        }    
    }
}

// IndexController.java
@Controllerpublic 
class IndexController {    
    @Autowired    
    private UserMapper userMapper;    
    
    @Autowired    
    private CommentMapper commentMapper;    

    @ModelAttribute    
    Comment setComment() {        
        return new Comment();    
    }    
    
    @GetMapping("/homePage")    
    public String index(Model model) {        
        Comment comment = new Comment();        
        model.addAttribute("list", commentMapper.showAllComment());        
        return "index";    
    }    
    
    @PostMapping("/homePage")    
    public String listCommentByName(Model model) {        
        model.addAttribute("list", commentMapper.showAllComment());        
        return "index";    
    }    
    
    @GetMapping("/update")    
    public String toUpdate(Model model) {        
        Comment comment = new Comment();        
        model.addAttribute("comment", comment);        
        return "index";    
    }    
    
    @PostMapping("/add")    
    public String addComment(Comment comment, HttpServletRequest request, RedirectAttributes attributes) {        
        // 通过 token 获取用户名        
        User user = new User();        
        Cookie[] cookies = request.getCookies();        
        for (Cookie cookie : cookies) {            
            if (cookie.getName().equals("token")){                
                String token = cookie.getValue();                
                user = userMapper.findByToken(token);                
                break;            
            }        
        }        
        comment.setUsername(user.getUsername());        
        Date date = new Date(System.currentTimeMillis());        
        comment.setTime(date);        
        commentMapper.addComment(comment);        
        return "redirect:/homePage#comment";    
    }
}
